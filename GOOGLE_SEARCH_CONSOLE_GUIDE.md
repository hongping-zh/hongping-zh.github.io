# Google Search Console 设置指南

## 🚀 立即执行步骤

### 1. 访问 Google Search Console
打开：https://search.google.com/search-console

### 2. 添加属性
- 点击 "添加属性"
- 选择 "网址前缀"
- 输入：`https://hongping-zh.github.io/`
- 点击 "继续"

### 3. 验证所有权
**推荐方法：HTML文件验证**
1. 下载 Google 提供的验证文件
2. 上传到仓库根目录（替换现有的 google-site-verification.html）
3. 或者使用已创建的文件，在 Google 中选择 "HTML文件" 方法

### 4. 提交 Sitemap
验证完成后：
1. 进入左侧菜单 "Sitemaps"
2. 在 "添加新的站点地图" 中输入：`sitemap.xml`
3. 点击 "提交"

### 5. 请求索引
1. 使用 "网址检查" 工具
2. 输入：`https://hongping-zh.github.io/`
3. 点击 "请求编入索引"

## 📊 监控指标

设置完成后，关注以下报告：
- **效果**：点击次数、展示次数、点击率、平均排名
- **索引**：已编入索引的网页数量
- **增强功能**：结构化数据、搜索结果中的丰富结果

## ⏱️ 预期时间线

- **1-3天**：Google 开始抓取网站
- **1周内**：开始在搜索结果中出现
- **2-4周**：主要关键词排名稳定
