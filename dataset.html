Dataset and Reproducibility

This page summarizes the public dataset, metadata, and reproducibility materials for the SSRN preprint on FP16, NF4, and INT8 LLM inference energy.

Model-size crossover

The primary measured sign-change bracket is 1.5B--6.0B. Analytical zero crossings are exploratory interpolation descriptors, not independently validated hardware constants.

  • RTX 4090D NF4, batch size 8: analytical N* = 2.54B.
  • RTX 4090D INT8, batch size 8: analytical N* = 3.45B.
  • RTX 5090 NF4, batch size 1 metadata: analytical N* = 4.42B.

Cloud GPU measurement caveat

Cloud GPU measurements may be affected by virtualization, vGPU/container isolation, power caps, thermal state, shared-infrastructure noise, and possible physical-GPU-level NVML telemetry. The analysis therefore emphasizes within-platform relative changes against FP16 baselines.
